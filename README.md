# Founcode

> Blueprint → Plan → Execute → Verify. From an idea to a verified app — natively on Windows.

Founcode is a Windows-first desktop app that turns AI coding agents (Claude Code, and soon Codex, Gemini CLI, OpenCode) into a disciplined engineering workflow. Start from a **plain-language idea** (or an existing half-built project) and Founcode drafts a PRD and a task plan (**Blueprint**), then works each task through a reviewable **plan**, an isolated git **worktree**, and independent **verification** before anything touches your branch.

**Status: v0.5 — Blueprint + Plan/Execute/Verify complete; heading to packaging & licensing (Phase 6).**

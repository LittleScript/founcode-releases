# Founcode

> Plan → Execute → Verify. Your AI coding agents, orchestrated — natively on Windows.

Founcode is a Windows-first desktop app that turns AI coding agents (Claude Code, and soon Codex, Gemini CLI, OpenCode) into a disciplined engineering workflow: every task starts with a reviewable plan, executes in an isolated git worktree, and ends with independent verification before anything touches your branch.

**Status: pre-alpha — in active development.**
